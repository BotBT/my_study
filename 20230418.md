### bool 자료형
- Ture(참)
- False(거짓)

if조건 :
    True(참)일 때만 안쪽 코드블럭을 실행
else :
    True가 아닐때 실행하려는 코드
    else는 조건이 False(거짓)일 때 안쪽 코드 블럭을 실행

if 조건1:
    조건1 이 참일 때 실행
else 조건2:
    조건1은 False, 조건2는 True일때 실행
else :
    조건1 False 조건2 False 일때 실행
